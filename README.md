# Pollinations AI 聊天助手

## 项目简介
Pollinations AI 聊天助手是一个基于网页的人工智能对话平台，旨在为用户提供方便、直观的聊天体验。通过整合多种先进的 AI 模型，用户可以进行多轮对话、上传文件或图片，并即时获得智能回复。

这个项目专注于：

- 简单易用的 UI / UX 设计  
- 支持多模型选择（GPT、Mistral、Llama、Claude、Gemini）  
- 直观的文件上传与预览功能  
- Markdown 支持与代码高亮  
- 完全在浏览器运行，无需安装桌面应用程序  

---

# 功能特色

## 1. 多模型支持

| 模型名称 | 说明 |
|---------|------|
| GPT-4o | OpenAI 最新模型，适合多轮对话与复杂问题 |
| Mistral Large | 开源大型模型，快速生成回复 |
| Llama 3 | Meta 提供的 LLM，适合知识问答 |
| Claude 3 | Anthropic 产品，偏向安全与伦理回答 |
| Gemini Pro | Google Gemini 系列，支持广泛应用场景 |

---

## 2. 即时消息与 Markdown 支持

- 消息支持文字、图片、代码片段  
- Markdown 自动渲染  
- 代码高亮显示  
- 内置复制代码按钮，方便程序分享  

---

## 3. 文件上传与预览

- 支持多种文件类型（图片、文本文件、代码文件）  
- 自动生成文件预览缩略图  
- 支持拖放或点击上传  
- 上传的文本文件可直接作为聊天参考  

---

## 4. 用户体验设计

- 完整响应式设计（手机 / 桌面）  
- 聊天气泡自动调整宽度  
- 自动滚动到最新消息  
- 模拟打字动画显示 AI 思考  
- 可清除单个或所有对话  

---

## 5. 记录与存储

- 对话记录自动保存到 LocalStorage  
- 刷新网页仍可继续聊天  
- 可自定义对话标题管理多个聊天  

---

# 技术架构

- 前端框架：HTML + CSS + JavaScript  
- UI 框架：TailwindCSS  
- 代码高亮：Highlight.js  
- Markdown 解析：Marked.js  
- AI 通讯：Pollinations API  

---

# 使用方式

1. 打开网页  
2. 创建或选择对话  
3. 在输入框输入消息  
4. 上传文件或图片并点击 发送  
5. AI 回复会即时显示  

> 无需注册账号即可使用（部分功能需要浏览器支持 LocalStorage）

---

# 注意事项

- AI 回复可能存在错误，请自行判断重要信息  
- 建议使用现代浏览器以获得完整功能  
- 文件上传大小受浏览器与网页配置限制
